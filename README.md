# tstart

Quick starter for typescript project.

Includes babel + typescript, eslint, tslint, prettier.

tstart works on macOS, Windows, and Linux.

## Get started

Using npx:

```bash
npx tstart init my-app
```

Using npm / yarn:

```bash
npm i -g tstart@latest # Or yarn global add tstart@latest
tstart init my-app
```

## Other Commands

```bash
tstart --help
```
